# PRODIGY_WD_05 – Weather Application

## 📖 Description

A responsive Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using the OpenWeather API. The application features a modern user interface with dynamic weather updates, detailed weather information, and an intuitive user experience.

## ✨ Features

- 🌍 Search weather by city name
- 📍 Current location weather
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 💨 Wind speed details
- 🌅 Sunrise and sunset timings
- 🌤️ Weather condition icons
- 🌙 Dark and Light theme toggle
- 📱 Fully responsive design
- ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
```

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/RithuParna19/PRODIGY_WD_05.git
```

2. Open the project folder.

3. Add your OpenWeather API key in `script.js`.

4. Open `index.html` in your browser.
